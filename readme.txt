== Description ==

A plugin that allows you to organize ideas for future blog posts, plugins, anything at all in one simple page under Manage > Ideas.

== Installation ==

1. Upload the entire folder of the plugin ('wp-ideas' folder) to your plugins directory ('/wp-content/plugins/' folder). 
1. Activate the plugin through the 'Plugins' menu in Wordpress.
1. Go to the Manage > Ideas page to add or delete ideas.
